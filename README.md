WebIM-for-ThinkSNS
=====================

Webim plugin for ThinkSNS V3

依赖
====

*	MySQL版本不低于5.1
*	需要PHP版本不低于5.3.10
*	PHP访问外部网络，WebIM连接时需要访问WebIM服务器, 请确保您的php环境是否可连接外部网络, 设置php.ini中`allow_url_fopen=ON`.
*	ThinkSNS V3.x


安装
=====

*   插件解压到addons/plugin/目录
*   后台管理启动WebIM插件
*   WebIM管理界面配置相关参数  

源码
=====

ThinkSNS_Plugin.php: 与ThinkSNS集成类，读取ThinkSNS的用户关系、群组关系、通知信息
